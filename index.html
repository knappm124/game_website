What is it?

Game Master is a one stop shop for playing games. Both board games and card games can require additional items that are easy to lose or get damaged. has all that and more to help make your games faster and easier.

What can it do?

Game Master puts some of the most popular games at your fingertips. has pre-made game pages for classic games like Clue, Monopoly, and Scrabble, as well as newer games like Settlers of Catan and Ticket to Ride. Each game page has a link to the rules for the game and any necessary items like timers, dice, and scorecards.

Not sure what to play?

Go to the games list and filter by options like number of players, type of game, or game length in order to find the right game for you.
